发行版选择

目前流行的Linux发行版包括:

  • Fedora
  • Ubuntu
  • Linux Mint
  • Debian
  • CentOS
  • OpenSUSE
  • Gentoo
  • ArchLinux

哪些因素定义了一个发行版:

  • 包管理器: 安装、升级、卸载软件的管理器
  • 桌面环境: gnome, kde, xfce, lxde, unity, mate...
  • 稳定性: 稳定 or 新版本?
  • 社区支持: 文档、FAQ

推荐新手使用Ubuntu，已有经验的用户尝试使用ArchLinux。
